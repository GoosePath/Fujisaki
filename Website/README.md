## 管理する情報要素

### 1. サイト構造情報
- **URL階層**: ディレクトリ構造とパス体系
- **ナビゲーション**: メニュー構造と導線設計
- **パラメータ体系**: カテゴリーID、商品コード、検索パラメータ

### 2. カテゴリー情報
- **主要カテゴリー**: フード、ライフスタイル、ギフト、コスメ、宮城・東北
- **サブカテゴリー**: 各主要カテゴリー配下の詳細分類
- **カテゴリーID**: システム内部の識別子体系

### 3. キャンペーン・特集情報
- **定期キャンペーン**: サマーギフト、年末年始等
- **セール情報**: 月末セール、特別割引等
- **特集ページ**: 地域特産品、ブランド特集等

### 4. ページ機能・設計
- **検索・フィルタ機能**: ソート、絞り込み、表示形式
- **ユーザー機能**: 会員登録、カート、決済
- **レスポンシブ対応**: モバイル・デスクトップ対応状況
