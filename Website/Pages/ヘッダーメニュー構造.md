# ヘッダーメニュー構造詳細

## トップレベルナビゲーション（ヘッダーメニュー）

### 1. TOP
- **URL**: https://fujisaki-online.jp/
- **機能**: ホームページ
- **内容**: キャンペーン情報、特集、商品ランキング、注目キーワード

### 2. 宮城・東北の名産品
- **URL**: https://fujisaki-online.jp/sp.html?fkey=miyagitohoku
- **概要**: 地域特産品専門ページ
- **特集内容**:
  - 仙台小箱（自由に選べる仙台の名産品セット）
  - 牛たんselection（藤崎オリジナルや名店の牛たん）
  - 仙台の名産品（仙台を代表する名産品、自慢の逸品）
  - 仙台の銘菓・スイーツ
  - みやぎ酒蔵めぐり
  - 笹かま・福ふぐ・笹たわ（宮城の海産品）

### 3. フード
- **URL**: https://fujisaki-online.jp/goods/list.html?cid=food
- **概要**: 食品全般カテゴリー
- **主要サブカテゴリー**: 14の基本カテゴリー + 100以上のブランド別カテゴリー
- **詳細**: 別途「フードカテゴリー構造.md」参照

### 4. ライフスタイル
- **URL**: https://fujisaki-online.jp/goods/list.html?cid=life
- **概要**: 生活雑貨・インテリア・ファッション・美容関連
- **主要サブカテゴリー**:
  - ビューティー＆ウェルネス: `?cid=life-B_W`
  - ブークブラン: `?cid=life-bouquetblanc`
  - ファッション: `?cid=life-01`
  - リビング: `?cid=life-02`
  - ベビー・キッズ: `?cid=life-03`

### 5. ギフト
- **URL**: https://fujisaki-online.jp/goods/list.html?cid=gift
- **概要**: 贈り物・ギフト商品専門カテゴリー
- **主要サブカテゴリー**:
  - カテゴリーから選ぶ: `?cid=gift-01`
  - お祝いを贈る: `?cid=gift-02`
  - お返しを贈る: `?cid=gift-03`
  - お香典返し・お供え: `?cid=gift-04`
  - 藤崎のご贈答品カタログ: `?cid=gift-05`
  - カタログギフト: `?cid=gift-06`
  - 価格から選ぶ: `?cid=gift-07`

### 6. コスメ
- **URL**: https://fujisaki-online.jp/cosme/
- **概要**: 化粧品・美容関連商品（独立したサブドメイン構造）
- **基本カテゴリー**:
  - スキンケア
  - メイクアップ
  - ボディケア
  - ヘアケア
  - フレグランス
  - サプリメント・インナーケア

## ヘッダーメニューの設計特徴

### メニュー階層
```
ヘッダーメニュー（6項目）
├── TOP（ホーム）
├── 宮城・東北の名産品（地域特色）
├── フード（商品カテゴリー）
├── ライフスタイル（商品カテゴリー）
├── ギフト（用途別カテゴリー）
└── コスメ（商品カテゴリー）
```

### 分類軸の混在
1. **用途軸**: ギフト（贈答用途）
2. **商品軸**: フード、ライフスタイル、コスメ（商品種別）
3. **地域軸**: 宮城・東北の名産品（産地別）
4. **機能軸**: TOP（サイト機能）

### URL構造の特徴
- **統一パターン**: `/goods/list.html?cid=[カテゴリーID]`
- **例外パターン**:
  - TOP: `/`（ルート）
  - 宮城・東北: `/sp.html?fkey=miyagitohoku`（特集ページ）
  - コスメ: `/cosme/`（独立ディレクトリ）

## サブメニュー構造の特徴

### フードカテゴリー
- **3階層構造**: 大カテゴリー → 中カテゴリー → 小カテゴリー
- **ブランド軸の強さ**: 100以上の個別ブランドページ
- **特集・セールの常設化**: 季節商品が恒常カテゴリー化

### ライフスタイルカテゴリー
- **4階層構造**: 用途別 → 商品別 → ブランド別 → 価格別
- **最も複雑な構造**: 分類軸が最も多様
- **専門性の高いサブカテゴリー**: ビューティー&ウェルネス等

### ギフトカテゴリー
- **3軸分類**: シーン別 × 商品別 × 価格別
- **用途特化設計**: 贈答シーンに特化した分類
- **価格軸の重要性**: 予算別選択が重視される構造

## 課題と改善点

### 現在の課題
1. **分類軸の不統一**: メニュー項目の分類基準が混在
2. **階層の不整合**: カテゴリーごとに異なる階層深度
3. **重複構造**: 同一商品が複数カテゴリーに存在
4. **専門性のバラつき**: カテゴリーごとの詳細度に差

### 改善提案
1. **MECE原則の適用**: 漏れなく・ダブりなくの分類
2. **階層統一**: 3階層での統一的な構造設計
3. **分類軸の整理**: 商品軸とシーン軸の明確な分離
4. **ユーザビリティ向上**: 直感的なナビゲーション設計
