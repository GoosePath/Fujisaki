# サイト回遊性向上プロジェクト

## タスク概要
藤崎オンラインストアの回遊性を高め、ユーザーが目的とする商品に辿り着きやすい状態を作ることを目的としたプロジェクト。

## 目的
キャンペーン以外の目的でサイトに入ってきたユーザーが、サイトを回遊して目的の商品に辿り着き、購買へとつなげやすい状態を作る。

## 実行フェーズ
1. **フェーズ1**: 情報設計の改善（現状のメニュー・キャンペーンの分類・整理）
2. **フェーズ2**: UIデザインの統一（ナビゲーションのレイアウト・コンポーネント修正）
3. **フェーズ3**: 実装ディレクション（ベンダー要件書作成・実装管理）

## 成功指標（KPI）
- トップページから常設カテゴリページへの遷移率: 5-14% → 20%以上
- サイト全体の平均ページビュー数: 現状値から30%向上
- 通常期のコンバージョン率: 現状値から20%向上
