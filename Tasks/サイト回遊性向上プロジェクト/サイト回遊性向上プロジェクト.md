# サイト全体の回遊性を高め、ユーザーが目的とする商品に辿り着きやすい状態を作る

## 背景

現在の藤崎オンラインストアはキャンペーンの表示に特化しており、それに対して特定の商品やカテゴリーの検索や探索が難しく、サイト内の回遊性が低い状態にある。これがキャンペーン以外の期間の売り上げが低い要因の一つとなっている。

### 具体的な問題点
現在の藤崎オンラインストは、総じて目的とする商品に到達できない / 到達までに必要なステップが多い状態となっている。

要因
1. **カテゴリー探索導線の混在**
   - SEARCH内のカテゴリー、ヘッダーナビゲーションのカテゴリ、 MENU内のカテゴリー で表示されている項目やクリックした歳のインタラクションが異なり、混乱が生じやすくなっている
   - 例：
      - SEARCH内 :
         - 項目 : 夏のギフト / 宮城・東北の名産品 / フード・スイーツ・リカー / ライフスタイル / ギフト / コスメ
         - 導線 : クリックすると小カテゴリーの一覧へと遷移する
      - ヘッダーナビゲーション内 : 
         - 項目 : TOP / 宮城・東北の名産品 / フード / ライフスタイル / ギフト / コスメ
         - 導線 : クリックすると各カテゴリーページへと遷移する

2. **キーワード検索へのアクセスがしづらい**
   - 検索ボックスが SEARCH メニュー内に入っているため、検索を開始するために１クリックが必要。見つけづらい状態となっている。
   - SEARCH は画面左端上部に表示されているが、これは現在の一般的なサイトデザインではあまり見ない構成のため、サイト訪問者にとって馴染がなく、扱いづらい可能性が高い。

3. **表記の不統一**
   - 項目表記の揺れ：一部の項目は、英字・日本語で表記が混在している
   - 例 : 宮城・東北の名産品 vs MIYAG&TOHOKU,  FOOD vs フード vs フード・スイーツ・リカー

### データ分析結果

前提 : 全サイト訪問者のうち 20% はトップページから流入しており、これが流入ページ別では最も多いため、トップページからカテゴリー/キャンペーンページへの遷移率で現在のパフォーマンスを比較する。

- 夏ギフト :  " トップページをLPとした時のルート別で見ると、トップページから夏ギフトページへは、ページ遷移率（24.7%）・コンバージョン率（67.2%）となっている
- 常設カテゴリページ : 常設のカテゴリページは、トップページからカテゴリーページへの遷移率が5% - 13%の間と10%以上の開きがある

メニューの導線設計の改善によって常設カテゴリーページへの遷移率を向上する余地がある

### 他の類似サイトの比較：三越伊勢丹オンラインストア

**URL**: https://www.mistore.jp/shopping

**ベストプラクティス**
- ナビゲーションの基本的な原則に則っており、目当ての商品を探しやすい・サイトを回遊しやすい
- 異なるカテゴリーは異なるセクションとしてはっきりと分かれている（2つ以上のカテゴリーが混在しない・セクション間に異色のボーダーを入れる）
- ナビゲーションメニューは、大項目・それに属する小項目で階層立てて並べており、見やすい

## 目標
サイトの検索 / ナビゲーション の表示を改善し、サイトの訪問者が、目的の商品に辿り着き、購買をしやすい状態を作る。

## 対応事項
1. [SEARCH / ヘッダーナビゲーション を統合する ]()
   - ヘッダーに検索ボックスを常時表示、ヘッダーからカテゴリーの一覧の閲覧と小項目への絞り込みを行えるようにする。
2. [トップページの FOOD / LIFE STYLE / GIFT / COSME / MIYAGI&TOHOKU のセクションの表記・表示カテゴリーを統合されたヘッダーナビゲーショの表示と統一する。]()
- [MENU内の項目を統一されたヘッダーナビゲーションの表示に合わせて修正する。]()
