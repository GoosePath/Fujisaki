# 藤崎オンラインストア ヘッダー実装要件書

## 1. プロジェクト概要

### 目的
藤崎オンラインストアのヘッダーナビゲーションを刷新し、ユーザーの商品発見性とサイト回遊性を向上させる。

### 対象
- AIによるモックアップ作成
- 開発チームによる実装

### モックアップ
- **デザインモックアップURL**: https://v0.dev/chat/mobile-header-design-BImQVy9Bf3G
レスポンシブ対応済みのため、モバイルで上記リンクを開くことで、モバイル版のレイアウトも確認可能。

### 基本方針
- 検索ボックスを常時表示
- デスクトップ：メガメニュー方式
- モバイル：検索ボックス連動型カテゴリー表示

## 2. 現状サイトからの変更箇所

### 2.1 変更しない項目（現状維持）
- **MENU**: 画面左上にピンク背景で表示されているサイドメニューはそのまま残す
  - ハンバーガーメニューアイコン
  - メニュー開閉機能
  - サイドメニュー内のコンテンツ構成

### 2.2 削除する項目
- **サイドメニュー下部のSEARCHメニュー**: 現在サイドメニューの下に表示されている検索機能（黒い背景）を削除
  - サイドメニューのSearch項目
  - 既存の検索ボックス配置

### 2.3 新規追加・統合する項目
機能の挙動の詳細はプレビューを参照

- **ヘッダー検索ボックス**: 既存のSEARCH内の検索ボックスをヘッダーに統合
- **ヘッダーメニュー**: 
  - デスクトップ : 
  　　- 既存のSEARCHメニュー内に表示されている項目を既存のヘッダーのメニューに統合。
     - **メガメニュー**: ホバー時に表示されるメガメニュー機能を実装。これで既存のSEARCHメニュー内に大カテゴリー -> 小カテゴリー への絞り込みを代替できる。
  - モバイル : 
    - メニュー項目 : ヘッダーから表示を削除。サイドメニュー内のアイテムと被っているため、また下記の検索ボックス連動カテゴリー表示で代替可のため。
    - 検索ボックス連動カテゴリー表示**: 検索ボックスタップ時の既存のSEARCHメニュー内に表示されているカテゴリー項目を検索ボックス下に表示する。ここでは大カテゴリ- -> 小カテゴリーへの絞り込みが可能

## 3. 機能要件

### 3.1 共通要件（デスクトップ・モバイル共通）

#### 検索ボックス
- **表示**: ヘッダー内に常時表示
- **機能**: 商品検索機能（キーワード検索） (既存のもと同様)
- **デザイン**: 検索アイコン付き、プレースホルダーテキスト表示
- **位置**: デスクトップの場合はヘッダー左上部、モバイル時はヘッダーのサイトロゴ下部, 現在 (宮城・東北の名産品.. ) ナビゲーションメニューが並んでいる位置

#### ロゴ (既存)
- **表示**: 藤崎オンラインストアのロゴ
- **機能**: クリックでトップページへ遷移
- **位置**: ヘッダー中央上部

#### ユーティリティメニュー (既存)
- **内容**: ログイン/新規登録、カート 
- **位置**: ヘッダー右端
- **詳細**: 後日定義

### 3.2 デスクトップ版要件

#### メインナビゲーション
- **表示方式**: 水平メニューバー
- **カテゴリー**: 5つのメインカテゴリーを常時表示
  1. 宮城・東北の名産品
  2. ギフト贈答
  3. フード・スイーツ・リカー
  4. コスメ美容
  5. ライフスタイル

#### メガメニュー仕様
- **トリガー**: マウスホバーで即座に表示
- **表示エリア**: 画面幅100%（フルワイド）
- **背景**: 白背景（または半透明オーバーレイ）
- **閉じる動作**: メガメニュー外へのマウスアウトで自動的に閉じる
- **アニメーション**: スムーズなフェードイン/アウト（200ms程度）

### 3.3 モバイル版要件

#### 検索ボックス連動カテゴリー表示
- **トリガー**: 検索ボックスをタップ
- **表示方法**: 検索ボックスの直下にメインカテゴリー一覧を表示
- **特徴**: 
  - 検索ボックスは表示したまま使用可能
  - カテゴリー選択と検索の両方が可能
- **閉じる動作**: 
  - カテゴリー選択時
  - 検索実行時
  - 画面の他の部分をタップ時

## 5. UI/UX仕様

### 5.1 視覚デザイン

#### カラースキーム
- **メインカラー**: 藤崎 ブランドカラー #D88395 に準拠
- **背景色**: 白またはライトグレー
- **テキスト色**: ダークグレー（#333333）
- **ホバー時**: ブランドカラー #D88395 アクセントカラー  #F2E4E9 を使う

#### タイポグラフィ
- **メインカテゴリー**: 14-16px、太字
- **サブカテゴリー**: 13-14px、標準
- **日本語フォント**: 游ゴシック、ヒラギノ角ゴ等

### 5.2 インタラクション

#### ホバーエフェクト
- **メインカテゴリー**: 下線表示またはカラー変更
- **サブカテゴリー**: 背景色変更または矢印アイコン表示
- **トランジション**: 200ms ease-in-out

#### メガメニュー表示
- **表示アニメーション**: フェードイン＋上からスライド
- **オーバーレイ**: 下層コンテンツに半透明マスク（任意）

### 5.3 レスポンシブブレークポイント
- **デスクトップ**: 1024px以上
- **タブレット**: 768px - 1023px
- **モバイル**: 767px以下

## 6. 技術仕様

### 6.1 実装技術
- **HTML5**: セマンティックマークアップ

- **JavaScript**: バニラJSまたはjQuery
