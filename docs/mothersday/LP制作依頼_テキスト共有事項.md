## 5. レスポンシブ挙動（PC / SP の違い）

画像だけでは判断しにくい動作仕様をまとめる。

| 対象 | PC（768px以上） | SP（767px以下） |
|------|----------------|-----------------|
| カテゴリ導線 | 4列グリッド | Swiperスライド（1.5枚表示、中央寄せ、自動再生3.5秒）+サムネイル4つ |
| ランキング | 5列グリッド（横1行） | 2列グリッド |
| 商品グリッド（通常） | 4列 | 2列 |
| 商品グリッド（お花のギフト） | 3列（max-width 75%で中央寄せ） | 2列（max-width 100%） |
| カテゴリーから選ぶ | 3列グリッド | 2列グリッド |
| メインビジュアル | PC画像を表示 | SP画像を表示 |
| コスメバナー | PC画像を表示 | SP画像を表示 |
