# 藤崎プロジェクト ファイル構成

## 概要
このディレクトリは株式会社藤崎様のECサイト分析・改善プロジェクトに関連するドキュメントを管理します。

## ファイル一覧

### プロジェクト管理
- `プロジェクト概要.md` - プロジェクトの全体像、契約内容、スケジュール管理
- `進捗管理.md` - 各タスクの進捗状況とマイルストーン管理
- `課題管理表.md` - 発見した課題と対応状況の一覧

### 分析関連
- `データ基盤レビュー.md` - Google Analytics等のデータ基盤の現状評価と改善提案
- `購買分析レポート_1回目.md` - 第1回目の定量分析レポート
- `購買分析レポート_2回目.md` - 第2回目の定量分析レポート
- `顧客セグメント分析.md` - 会員データのセグメント分析結果

### 改善提案
- `EC購買体験改善提案.md` - ECサイト全体の購買体験改善案
- `仙台小箱プロモーション戦略.md` - 仙台小箱の具体的なプロモーション施策
- `改善施策ロードマップ.md` - 改善施策の実行計画

### ミーティング記録
- `キックオフミーティング_20250602.md` - 勢田様とのキックオフミーティング記録
- `週次定例_YYYYMMDD.md` - 週次定例会議の記録（日付別）

### 参考資料
- `契約書_業務委託基本契約書.pdf` - 締結済み契約書のコピー
- `藤崎ECサイト_現状分析メモ.md` - 初期ヒアリング内容
